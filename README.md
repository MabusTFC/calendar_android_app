# Приложение “Расписание занятий”
Лабораторная работа Android приложение для просмотра учебного расписания.

## Функционал
-Просмотр расписания на текущий день
-Просмотр расписания на неделю по её четности 
-Автоматическое определение и ручное переключение четности недели

## Технологии
 * Kotlin
 * Android SDK
 * ViewBinding
 * RecyclerView

## Установка
### Требования
 * JDK 17+
 * Android SDK API 24+

- ### Шаги​
 1. Из корня проекта (для сборки и установки на подключенное устройство):

    ```bash
    .\gradlew installDebug
    ```
